# AIR INDIA – Flight Booking System

A simple flight booking website with OTP login, passenger details form, ticket preview, QR payment, payment success sound, and PDF download.

---

## Live Demo
https://flight-booking-sandy.vercel.app/

---

## Features
- OTP login using EmailJS  
- Passenger & travel details form  
- Automatic fare calculation  
- Ticket preview page  
- QR-based payment popup  
- Payment success animation + sound  
- Download ticket as PDF  

---

## File Structure

/project

---

## How to Run
1. Open **index.html** in your browser  
2. Enter email → Get OTP → Verify  
3. Fill booking details  
4. Preview the ticket  
5. Scan QR → Payment success  
6. Download PDF ticket
